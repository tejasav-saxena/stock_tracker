Stock Price Tracker

About the Project

The Stock Price Tracker is a Python-based application that uses Yahoo Finance data to display and compare stock prices for various companies. It provides real-time stock price updates and allows users to select multiple stocks to compare their performance over the past 30 days.

Features

  • View the latest stock price for a selected company
  • Compare stock prices for up to 3 companies on a graph
  • Interactive graph with data points and hover annotations
  • Toggle full-screen mode for a detailed view of the stock comparison graph

How to Use

Get Single Stock Price: Select a stock symbol from the dropdown menu, then click "Get Price" to see the latest price.
Compare Multiple Stocks: Select 2 or 3 stock symbols, then click "Compare Stocks" to view a comparison graph of their performance over the last 30 days.

Python Libraries Used

  • Yahoo Finance (yfinance) for retrieving stock data
  • Matplotlib for creating the comparison graph
  • Tkinter for the graphical user interface
  • mplcursors for interactive data points on the graph
